# Local Music Player

A simple local music player for macOS built with **Python** and **Qt (PySide6)**.

## Features

- Play local audio files (MP3, WAV, FLAC, etc.)
- Folder scanning
- Shuffle and repeat
- Mini player mode
- Album cover display
- Clean dark UI

## Download

Download the latest version from the **Releases** page:

➡️ https://github.com/slanacexe/Local-Music-Player-App/releases

Download the `.dmg` file and drag the app into **Applications**.

## Installation (from source)

```bash
git clone https://github.com/slanacexe/Local-Music-Player-App.git
cd Local-Music-Player-App
